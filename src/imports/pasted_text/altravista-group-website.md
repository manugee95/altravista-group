Create a premium, modern, and highly professional corporate website for "ALTRAVISTA GROUP", a diversified engineering, manufacturing, and construction group headquartered in Lagos, Nigeria.

The website should position Altravista Group as a parent company with several subsidiaries offering specialized services across engineering, construction, CNC manufacturing, facade systems, and industrial products.

The website will use:

Frontend:

* React
* TypeScript
* Tailwind CSS
* React Router
* Framer Motion

Backend:

* Headless WordPress CMS

API Integration:

* WordPress REST API (not GraphQL)

The website must be fully responsive and optimized for desktop, tablet, and mobile devices.
